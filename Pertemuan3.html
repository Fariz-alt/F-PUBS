Uji ANOVA Two-Way
(Dua Arah)

Menganalisis pengaruh dua faktor dan interaksinya pada satu variabel dependen.

Apa Itu Uji ANOVA Two-Way?

ANOVA Two-Way adalah perluasan dari ANOVA One-Way. Jika ANOVA One-Way digunakan untuk membandingkan rata-rata di antara dua atau lebih kelompok yang dibentuk oleh satu faktor kategorikal, maka ANOVA Two-Way digunakan ketika kita memiliki dua faktor kategorikal dan ingin melihat pengaruhnya terhadap satu variabel dependen kontinu.

Selain menguji efek individual dari setiap faktor (disebut efek utama atau main effects), ANOVA Two-Way juga memungkinkan kita untuk menguji efek interaksi (interaction effect) antara kedua faktor tersebut. Efek interaksi terjadi ketika efek dari satu faktor pada variabel dependen tidak sama di semua level faktor lainnya.

Kapan Menggunakan ANOVA Two-Way?

Anda harus menggunakan ANOVA Two-Way ketika:

  • Anda memiliki satu variabel dependen kontinu (misalnya, nilai ujian, tinggi tanaman, tekanan darah, dll.).
  • Anda memiliki dua variabel independen kategorikal (faktor), masing-masing dengan dua atau lebih level/kelompok (misalnya, Jenis Kelamin: Pria/Wanita; Jenis Obat: A/B/C; Tingkat Pendidikan: SMA/Sarjana/Pascasarjana).
  • Anda ingin mengetahui:
    • Apakah ada efek signifikan dari Faktor 1 pada variabel dependen?
    • Apakah ada efek signifikan dari Faktor 2 pada variabel dependen?
    • Apakah ada interaksi antara Faktor 1 dan Faktor 2 yang memengaruhi variabel dependen?

Hipotesis dalam ANOVA Two-Way

Dalam ANOVA Two-Way, ada tiga pasang hipotesis nol (\( H_0 \)) dan hipotesis alternatif (\( H_1 \)) yang diuji:

1. Untuk Efek Utama Faktor A:

  • \( H_0 \): Tidak ada perbedaan signifikan dalam rata-rata variabel dependen di antara level-level Faktor A.
  • \( H_1 \): Ada perbedaan signifikan dalam rata-rata variabel dependen di antara level-level Faktor A.

2. Untuk Efek Utama Faktor B:

  • \( H_0 \): Tidak ada perbedaan signifikan dalam rata-rata variabel dependen di antara level-level Faktor B.
  • \( H_1 \): Ada perbedaan signifikan dalam rata-rata variabel dependen di antara level-level Faktor B.

3. Untuk Efek Interaksi Faktor A x Faktor B:

  • \( H_0 \): Tidak ada interaksi signifikan antara Faktor A dan Faktor B yang memengaruhi variabel dependen (efek Faktor A konsisten di semua level Faktor B, dan sebaliknya).
  • \( H_1 \): Ada interaksi signifikan antara Faktor A dan Faktor B yang memengaruhi variabel dependen (efek Faktor A bervariasi di antara level-level Faktor B, atau sebaliknya).

Asumsi ANOVA Two-Way

Agar hasil ANOVA Two-Way valid, beberapa asumsi harus dipenuhi:

  • Variabel Dependen Kontinu: Variabel dependen harus diukur pada skala interval atau rasio.
  • Variabel Independen Kategorikal: Kedua variabel independen harus kategorikal (nominal atau ordinal).
  • Independensi Observasi: Setiap observasi (data point) harus independen satu sama lain.
  • Tidak Ada Outlier Signifikan: Data tidak boleh memiliki outlier ekstrem yang dapat memengaruhi hasil secara berlebihan.
  • Normalitas: Variabel dependen harus berdistribusi normal secara *aproksimasi* untuk setiap kombinasi kelompok dari kedua faktor.
  • Homogenitas Variansi (Homoscedasticity): Variansi variabel dependen harus kira-kira sama di antara semua kombinasi kelompok dari kedua faktor. Ini dapat diuji dengan **Levene's Test**.

Langkah-langkah Melakukan Uji ANOVA Two-Way di R

Mari kita gunakan contoh hipotetis. Misalkan kita ingin melihat apakah ada pengaruh jenis pupuk (Faktor A: Pupuk A, Pupuk B) dan jenis tanah (Faktor B: Tanah Liat, Tanah Pasir) terhadap tinggi tanaman (variabel dependen kontinu).

Data Contoh:

Jenis Pupuk Jenis Tanah Tinggi Tanaman (cm)
Pupuk ATanah Liat15, 17, 16, 18, 14
Pupuk ATanah Pasir10, 12, 11, 9, 13
Pupuk BTanah Liat20, 22, 19, 21, 23
Pupuk BTanah Pasir14, 16, 15, 17, 13

1. Membuat Data Frame di R

# Membuat data frame untuk contoh
data_tanaman <- data.frame(
  Tinggi_Tanaman = c(15, 17, 16, 18, 14, # Pupuk A, Tanah Liat
                     10, 12, 11, 9, 13,  # Pupuk A, Tanah Pasir
                     20, 22, 19, 21, 23, # Pupuk B, Tanah Liat
                     14, 16, 15, 17, 13),# Pupuk B, Tanah Pasir
  Pupuk = factor(rep(c("Pupuk A", "Pupuk B"), each = 10)),
  Tanah = factor(rep(c("Tanah Liat", "Tanah Pasir", "Tanah Liat", "Tanah Pasir"), each = 5))
)

2. Visualisasi Data (Opsional, tapi Sangat Disarankan)

Visualisasi dapat membantu Anda memahami distribusi data dan potensi interaksi sebelum melakukan uji statistik.

Box Plot per Kombinasi Faktor:

boxplot(Tinggi_Tanaman ~ Pupuk * Tanah, data = data_tanaman,
        main = "Distribusi Tinggi Tanaman berdasarkan Pupuk dan Tanah",
        xlab = "Kombinasi Pupuk dan Tanah",
        ylab = "Tinggi Tanaman (cm)",
        col = c("#ADD8E6", "#90EE90", "#FFD700", "#FFA07A"),
        border = "darkgray")

Gambar 1: Contoh Box Plot Tinggi Tanaman berdasarkan Jenis Pupuk dan Jenis Tanah.

Plot Interaksi:

Plot interaksi sangat penting untuk memahami apakah efek satu faktor bergantung pada level faktor lainnya. Jika garis-garis tidak paralel, ini mengindikasikan adanya interaksi.

# Membutuhkan paket 'ggplot2' atau fungsi 'interaction.plot' dasar
# install.packages("ggplot2") # Jika belum terinstall
library(ggplot2)

ggplot(data_tanaman, aes(x = Pupuk, y = Tinggi_Tanaman, color = Tanah, group = Tanah)) +
  stat_summary(fun = mean, geom = "point", size = 3) +
  stat_summary(fun = mean, geom = "line", size = 1) +
  labs(title = "Plot Interaksi: Tinggi Tanaman vs. Pupuk & Tanah",
       x = "Jenis Pupuk",
       y = "Rata-rata Tinggi Tanaman (cm)",
       color = "Jenis Tanah") +
  theme_minimal() +
  theme(plot.title = element_text(hjust = 0.5))

# Atau menggunakan fungsi dasar R 'interaction.plot'
interaction.plot(x.factor = data_tanaman$Pupuk,
                 trace.factor = data_tanaman$Tanah,
                 response = data_tanaman$Tinggi_Tanaman,
                 fun = mean,
                 type = "b", # both points and lines
                 col = c("blue", "red"),
                 pch = c(16, 17),
                 main = "Plot Interaksi: Tinggi Tanaman vs. Pupuk & Tanah",
                 xlab = "Jenis Pupuk",
                 ylab = "Rata-rata Tinggi Tanaman (cm)")

Gambar 2: Contoh Plot Interaksi antara Jenis Pupuk dan Jenis Tanah terhadap Tinggi Tanaman.

3. Memeriksa Asumsi Homogenitas Variansi (Levene's Test)

Penting untuk memeriksa asumsi homogenitas variansi menggunakan Levene's Test.

# install.packages("car") # Jika belum terinstall
library(car)
leveneTest(Tinggi_Tanaman ~ Pupuk * Tanah, data = data_tanaman)

Jika p-value dari Levene's Test lebih besar dari 0.05, asumsi homogenitas variansi terpenuhi.

4. Melakukan Uji ANOVA Two-Way

Gunakan fungsi `aov()` untuk menjalankan analisis ANOVA.

# Jalankan ANOVA Two-Way (termasuk efek interaksi)
model_anova <- aov(Tinggi_Tanaman ~ Pupuk * Tanah, data = data_tanaman)

# Tampilkan ringkasan hasil ANOVA
summary(model_anova)

5. Interpretasi Hasil

Output dari `summary(model_anova)` akan menampilkan tabel ANOVA. Fokus pada kolom `Pr(>F)` (nilai p-value) untuk setiap faktor dan interaksi.

Contoh Output (hipotetis):

            Df Sum Sq Mean Sq F value   Pr(>F)    
Pupuk        1 101.25  101.25    40.5 9.39e-06 ***
Tanah        1 151.25  151.25    60.5 7.97e-07 ***
Pupuk:Tanah  1   1.25    1.25     0.5     0.49    
Residuals   16  40.00    2.50                     
---
Signif. codes:  0 ‘***’ 0.001 ‘**’ 0.01 ‘*’ 0.05 ‘.’ 0.1 ‘ ’ 1

Langkah-langkah Interpretasi:

  1. Periksa Efek Interaksi (`Pupuk:Tanah`) terlebih dahulu.
    • Jika p-value interaksi < level signifikansi (misalnya 0.05): Ada efek interaksi yang signifikan. Ini berarti efek satu faktor pada variabel dependen (tinggi tanaman) bergantung pada level faktor lainnya. Dalam kasus ini, **efek utama tidak boleh diinterpretasikan secara langsung** tanpa mempertimbangkan interaksi. Lihat plot interaksi untuk memahami detailnya.
    • Jika p-value interaksi > level signifikansi (misalnya 0.05): Tidak ada efek interaksi yang signifikan. Efek satu faktor pada tinggi tanaman relatif konsisten di semua level faktor lainnya. Anda kemudian dapat melanjutkan untuk menginterpretasikan efek utama.
  2. Periksa Efek Utama (`Pupuk` dan `Tanah`) (Hanya jika interaksi tidak signifikan atau jika Anda memahami cara menginterpretasikan efek utama di hadapan interaksi).
    • Untuk Pupuk: Jika p-value Pupuk < 0.05, ada perbedaan signifikan.
    • Untuk Tanah: Jika p-value Tanah < 0.05, ada perbedaan signifikan.

6. Melakukan Post-Hoc Test (Jika Diperlukan)

Jika efek utama atau interaksi signifikan, Anda mungkin perlu tes *post-hoc* seperti **Tukey's HSD** untuk mengidentifikasi kelompok mana yang berbeda secara spesifik.

# Contoh Tukey HSD untuk semua kombinasi (jika interaksi signifikan)
TukeyHSD(model_anova, which = "Pupuk:Tanah")

Output Tukey HSD akan menunjukkan perbedaan rata-rata antar kelompok, interval kepercayaan, dan p-value yang disesuaikan.
