Analisis Grafis:
Box Plot & Scatter Plot

Memahami distribusi dan hubungan antar variabel melalui visualisasi data.

Pendahuluan Analisis Grafis

Analisis grafis adalah langkah penting dalam eksplorasi data. Dengan memvisualisasikan data, kita dapat mengidentifikasi pola, tren, outlier, dan hubungan yang mungkin tidak terlihat dari sekadar angka. Dua grafik yang paling umum dan powerful untuk tujuan ini adalah Box Plot dan Scatter Plot.

Kedua visualisasi ini memberikan wawasan fundamental tentang struktur dan karakteristik dataset Anda, membantu Anda membuat keputusan yang lebih tepat dalam analisis lanjutan atau pemodelan.

1. Box Plot (Diagram Kotak Garis)

Box plot, atau diagram kotak garis, adalah metode standar untuk menampilkan distribusi data berdasarkan lima ringkasan statistik: nilai minimum, kuartil pertama (Q1), median (Q2), kuartil ketiga (Q3), dan nilai maksimum. Diagram ini sangat berguna untuk membandingkan distribusi data di antara beberapa kelompok atau untuk mengidentifikasi outlier.

Fungsi Utama Box Plot:

  • Menunjukkan Distribusi Data: Memberikan gambaran cepat tentang sebaran data (simetris atau miring).
  • Mengidentifikasi Outlier: Titik-titik di luar "jangkauan pagar" (whiskers) seringkali dianggap sebagai outlier.
  • Membandingkan Kelompok: Sangat efektif untuk melihat perbedaan distribusi antar kategori.

Struktur Box Plot:

  • Kotak (Box): Merepresentasikan Interquartile Range (IQR), yaitu rentang antara kuartil pertama (Q1) dan kuartil ketiga (Q3). Bagian tengah kotak adalah median (Q2).
  • Garis Tengah (Median): Garis di dalam kotak yang menunjukkan nilai tengah data.
  • "Janggut" (Whiskers): Garis yang memanjang dari kotak hingga nilai minimum dan maksimum yang masih dalam batas wajar (biasanya \( 1.5 \times \text{IQR} \) dari Q1 dan Q3).
  • Titik (Outliers): Titik-titik individual di luar whiskers yang dianggap sebagai outlier.

Cara Membuat Box Plot di R

Kita akan menggunakan data inflasi dari contoh sebelumnya untuk membuat box plot. Untuk demonstrasi, mari kita buat box plot dari variabel 'Inflasi'.

# Membuat data frame untuk contoh
data_regresi <- data.frame(
  Ekspor = c(22320.71, 21319.76, 23413.90, 19280.37, 21706.53, 20598.95,
             20861.29, 21996.26, 20744.94, 22144.52, 21995.93, 22391.28),
  Kurs = c(14979.00, 15274.00, 14979.00, 14751.00, 15062.00, 15026.00,
           14969.00, 15239.00, 15083.00, 15916.00, 15526.00, 15416.00)
)
# Membuat Box Plot
boxplot(data_regresi)

Output dari kode di atas adalah sebuah box plot yang menggambarkan sebaran inflasi. Anda akan melihat median, kuartil, dan jangkauan data, serta apakah ada titik data yang dianggap sebagai outlier.

2. Scatter Plot (Diagram Pencar)

Scatter plot adalah jenis grafik yang digunakan untuk menampilkan hubungan antara dua variabel kuantitatif. Setiap titik pada scatter plot merepresentasikan sepasang nilai dari dua variabel, satu pada sumbu horizontal (X) dan satu pada sumbu vertikal (Y). Ini adalah alat yang sangat baik untuk mengidentifikasi pola, tren, dan potensi hubungan kausal.

Fungsi Utama Scatter Plot:

  • Mengidentifikasi Hubungan: Menunjukkan apakah ada hubungan antara dua variabel (positif, negatif, tidak ada, non-linear).
  • Mendeteksi Pola: Memungkinkan untuk melihat pola atau kelompok dalam data.
  • Melihat Outlier Bivariat: Titik-titik yang menyimpang jauh dari pola umum dapat menjadi outlier.
  • Mengevaluasi Asumsi Regresi: Penting untuk memeriksa linearitas dan homoskedastisitas sebelum membangun model regresi.

Cara Membuat Scatter Plot di R

Kita akan membuat scatter plot untuk melihat hubungan antara 'Kurs' (nilai tukar) dan 'Inflasi' dari dataset data_regresi.

data_regresi <- data.frame(
  Bulan = c("Januari", "Februari", "Maret", "April", "Mei", "Juni",
            "Juli", "Agustus", "September", "Oktober", "November", "Desember"),
  Inflasi = c(5.28, 5.47, 4.97, 4.33, 4.00, 3.52, 3.08, 3.27, 2.28, 2.56, 2.86, 2.61),
  M1 = c(2422174.00, 2403593.54, 2408418.91, 2472869.46, 2423317.96, 2466093.07,
         2438058.76, 2410850.87, 2482229.03, 2497710.30, 2510759.32, 2675333.28),
  Ekspor = c(22320.71, 21319.76, 23413.90, 19280.37, 21706.53, 20598.95,
             20861.29, 21996.26, 20744.94, 22144.52, 21995.93, 22391.28),
  Kurs = c(14979.00, 15274.00, 14979.00, 14751.00, 15062.00, 15026.00,
           14969.00, 15239.00, 15083.00, 15916.00, 15526.00, 15416.00)
)

# Membuat Scatter Plot antara Kurs dan Inflasi
plot(data_regresi$Kurs, data_regresi$Inflasi,
     main = "Scatter Plot: Inflasi vs. Kurs", # Judul grafik
     xlab = "Kurs (IDR)",                     # Label sumbu X
     ylab = "Inflasi (%)",                    # Label sumbu Y
     pch = 16,                                # Bentuk titik (lingkaran padat)
     col = "#007bff")                         # Warna titik

Output dari kode di atas adalah sebuah scatter plot yang menunjukkan hubungan antara Inflasi dan Kurs. Jika ada tren, garis regresi merah akan membantu memvisualisasikannya.
