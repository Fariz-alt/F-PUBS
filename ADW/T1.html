Laporan Analisis Peramalan Penjualan Harian

Menggunakan Metode ARIMA Box–Jenkins

1. Pendahuluan

Peramalan (forecasting) merupakan proses penting dalam manajemen persediaan karena dapat memberikan estimasi permintaan di masa mendatang. Informasi yang akurat mengenai jumlah barang yang akan terjual membantu pengambil keputusan dalam menentukan jumlah pesanan yang optimal, sehingga dapat menghindari risiko kelebihan atau kekurangan stok.

Penelitian ini bertujuan untuk memodelkan data penjualan harian suatu barang pada sebuah pasar swalayan dengan metode ARIMA Box–Jenkins. Data yang digunakan merupakan data penjualan harian (AMOUNT) selama 149 hari berturut-turut. Model yang diperoleh diharapkan memiliki akurasi yang baik serta memenuhi asumsi-asumsi diagnostik sehingga dapat digunakan untuk meramalkan penjualan pada periode berikutnya.

2.2 Metode Analisis

Tahapan analisis menggunakan metode ARIMA Box–Jenkins meliputi:

  • Identifikasi Model: Pemeriksaan stasioneritas melalui plot deret waktu dan Augmented Dickey–Fuller Test. Analisis plot ACF dan PACF untuk menentukan orde model.
  • Estimasi Parameter: Estimasi parameter model menggunakan metode Maximum Likelihood (ML).
  • Pemeriksaan Diagnostik: Uji signifikansi parameter, uji white noise dengan Box–Ljung Test, dan uji normalitas sisaan dengan Shapiro–Wilk Test.
  • Peramalan: Menghasilkan prediksi untuk 12 periode ke depan beserta interval kepercayaan.

3. Hasil dan Pembahasan

3.1 Tahap Identifikasi

Pemeriksaan awal dilakukan melalui plot deret waktu pada Gambar 1. Berdasarkan visualisasi, data penjualan tidak menunjukkan tren yang jelas dan variansi relatif konstan, sehingga secara visual data cenderung stasioner.

Gambar 1. Plot Deret Waktu Penjualan Harian

Karena pada plot data tidak membentuk tren dan variansinya tidak beragam maka data tersebut stasioner

Penjualan <- data$Zt
Hari <- data$t
ts.plot(Penjualan)

Hasil Augmented Dickey–Fuller Test menunjukkan nilai p-value = 0,01619 < 0,05, sehingga dapat disimpulkan bahwa data stasioner secara statistik.

Plot ACF menunjukkan pola penurunan secara eksponensial (dies down), sementara plot PACF terputus pada lag ke-1 (cut off after lag 1). Pola ini mengarah pada dugaan model ARIMA(1,0,0).

3.2 Estimasi Parameter

Hasil estimasi parameter model ARIMA(1,0,0) dengan metode Maximum Likelihood adalah sebagai berikut:

  • Koefisien AR(1) = 0,7519 (s.e. = 0,0535)
  • Intersep = 1007,3008 (s.e. = 3,3302)
  • Varians sisaan (${\sigma}^2$) = 105,7
  • Nilai AIC = 1124,11

Nilai parameter AR(1) yang signifikan menunjukkan bahwa model ini mampu menangkap pola autokorelasi pada data.

3.3 Pemeriksaan Diagnostik

  • Uji Signifikansi Parameter: p-value seluruh parameter ≤ 0,05, menunjukkan parameter signifikan secara statistik.
  • Uji White Noise (Box–Ljung Test): p-value = 0,7119 (> 0,05), menunjukkan bahwa sisaan tidak memiliki autokorelasi yang tersisa.
  • Uji Normalitas Sisaan (Shapiro–Wilk Test): p-value = 0,7323 (> 0,05), menunjukkan sisaan berdistribusi normal.

Berdasarkan hasil uji ini, model **ARIMA(1,0,0)** memenuhi seluruh kriteria diagnostik.

6. Kesimpulan

Berdasarkan hasil analisis, diperoleh kesimpulan sebagai berikut:

  • Model terbaik yang sesuai untuk data penjualan harian adalah **ARIMA(1,0,0)**.
  • Model yang diperoleh memenuhi asumsi stasioneritas, signifikansi parameter, white noise, dan normalitas sisaan.
  • Hasil peramalan menunjukkan bahwa jumlah penjualan harian selama 12 hari ke depan berada pada kisaran 1007–1021 unit dengan fluktuasi yang relatif kecil.
  • Model ini dapat digunakan sebagai acuan dalam pengambilan keputusan terkait pengelolaan persediaan barang di masa mendatang.
