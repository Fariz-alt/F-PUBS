Metode ARIMA Box-Jenkins untuk Analisis Deret Waktu

Model Autoregressive Integrated Moving Average (ARIMA) digunakan untuk memahami dan memprediksi data deret waktu.

Pengenalan Model ARIMA

Model ARIMA (Autoregressive Integrated Moving Average) diperkenalkan oleh George Box dan Gwilym Jenkins. Model ini menggabungkan dua aspek utama: autoregresive (AR) dan moving average (MA).

Secara umum, model ini dinotasikan sebagai ARIMA (p, d, q), di mana:

  • p: Orde dari proses autoregressive (AR).
  • d: Derajat differencing yang dibutuhkan untuk membuat data stasioner.
  • q: Orde dari proses moving average (MA).

Empat Tahap Pendekatan Box-Jenkins

Analisis dengan metode Box-Jenkins melibatkan empat tahap utama yang saling berulang hingga model yang optimal ditemukan.

Tahap 1: Identifikasi Model Awal

Tahap ini bertujuan untuk menentukan apakah data deret waktu sudah stasioner dan memilih model ARIMA (p, d, q) yang potensial.

a. Pemeriksaan Stasioneritas

Stasioneritas data diperiksa melalui plot time series. Data dikatakan stasioner jika rata-rata (mean) dan variansi (variance) konstan sepanjang waktu. Jika data tidak stasioner, langkah-langkah berikut dapat dilakukan:

  • Jika non-stasioner terhadap rata-rata, lakukan proses differencing (pembedaan).
  • Jika non-stasioner terhadap variansi, lakukan transformasi Box-Cox.
  • Jika non-stasioner dalam keduanya, stabilkan variansi terlebih dahulu, lalu lakukan differencing.

Berbagai Pola Stasioneritas pada Plot Time Series

Stasioner terhadap rata-rata maupun variansi

Non-Stasioner terhadap rata-rata tapi stasioner terhadap variansi

Stasioner terhadap rata-rata tapi Non-stasioner terhadap variansi

Non- Stasioner terhadap rata-rata maupun terhadap variansi

b. Analisis ACF dan PACF

Setelah data stasioner, kita menganalisis plot Autocorrelation Function (ACF) dan Partial Autocorrelation Function (PACF) untuk menentukan orde model (p dan q). Pola pada plot ini akan menunjukkan apakah model lebih cenderung ke arah AR, MA, atau kombinasi keduanya.

Catatan: Interpretasi pola ACF dan PACF memerlukan pemahaman mendalam tentang karakteristik model AR dan MA.

Tahap 2: Penaksiran Parameter

Setelah mengidentifikasi model sementara, langkah selanjutnya adalah menaksir parameter-parameter model. Sebagai ilustrasi, kita ambil contoh model AR(1).

Model umum ARIMA(p,d,q):

$$\left( 1 - \sum_{i=1}^{p} \phi_i B^i \right) (1 - B)^d Z_t = \left( 1 + \sum_{j=1}^{q} \theta_j B^j \right) \varepsilon_t$$

Di mana \(B\) adalah operator lag. Sebagai contoh, \(B^k Z_t = Z_{t-k}\).

Jika diasumsikan model sementara adalah ARIMA(1,0,0), maka \(p = 1\), \(d = 0\), dan \(q = 0\).

Model AR(1) dapat ditulis sebagai:

$$(1 - \phi_1 B) Z_t = \varepsilon_t$$

$$Z_t - \phi_1 Z_{t-1} = \varepsilon_t$$

atau

$$Z_t = \phi_1 Z_{t-1} + \varepsilon_t$$

Pertanyaan Penting: Apa yang terjadi jika parameter \(\phi_1\) bernilai satu?

Jika \(\phi_1 = 1\), maka model menjadi:

$$Z_t = Z_{t-1} + \varepsilon_t$$

Model ini dikenal sebagai Random Walk dan merupakan contoh deret waktu non-stasioner. Hal ini menunjukkan pentingnya pemeriksaan stasioneritas dan nilai parameter sebelum menggunakan model.

Tahap 3: Pemeriksaan Diagnostik

Tahap ini adalah validasi model. Ada dua pemeriksaan utama:

  • Uji Kesignifikanan Parameter

    Menggunakan Uji-t untuk memastikan bahwa parameter model yang diestimasi (misalnya $\phi$ dan $\theta$) signifikan secara statistik (berbeda dari nol).

  • Uji Kesesuaian Model

    Memeriksa asumsi sisa (residual) model. Sisa harus berdistribusi normal dan bersifat white noise (tidak ada korelasi antar sisa). Ini diuji dengan:

    • Uji Ljung-Box: Menguji apakah sisa bersifat white noise.
    • Uji Kolmogorov-Smirnov: Menguji apakah sisa berdistribusi normal.

Tahap 4: Peramalan (Forecasting)

Jika semua parameter model signifikan dan semua asumsi terpenuhi pada tahap diagnostik, maka model siap digunakan untuk peramalan nilai masa depan.

Namun, jika ada parameter yang tidak signifikan atau asumsi tidak terpenuhi, kita harus kembali ke Tahap 1 (Identifikasi Model Awal) untuk mencoba model ARIMA (p, d, q) yang lain.
